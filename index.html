Dynamic Client Configuration Setup Guide:

To avoid Google OAuth login 404 errors, you must register a Web Application Client in your own Google Cloud developer Console.

  1. Navigate to Google Cloud Console.
  2. Create a new project and search for the Google Drive API, then enable it.
  3. Go to the APIs & Services > Credentials page, click Create Credentials, and select OAuth client ID.
  4. Select Web application as application type.
  5. Set your Authorized Redirect URI to match your web deployment, or set it to: smartcashmemo://oauth.
  6. Copy and paste your generated Client ID and Client Secret below to instantly activate authentic backups!
